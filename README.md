## рҹ“Ӣ About The Project

**FastBite POS** is a fully functional Point of Sale desktop application built for a fast food restaurant. It was developed as a semester project for the **Advanced Programming (.NET)** course at **Islamia University of Bahawalpur**.

The application allows restaurant staff to manage the menu, register customers, place orders, and view business analytics through an interactive dashboard вҖ” all backed by a **SQL Server** database using **ADO.NET**.

---

## вңЁ Features

### Core Features
- рҹ“Ҡ **Dashboard** вҖ” Real-time stats and 3 interactive charts
- рҹҚ” **Menu Management** вҖ” Full CRUD for food items with category and status
- рҹ‘Ө **Customer Management** вҖ” Register and manage customer profiles
- рҹ§ҫ **Order Management** вҖ” Create orders, pick customers, add items, track status

### Bonus Features (All 6 Implemented)
- рҹ”Қ **Search & Filter** вҖ” Text search with dropdown filters on every view
- рҹ“Ҳ **Dashboard Charts** вҖ” Bar, Pie and Line charts with live database data
- вҡЎ **Async Loading** вҖ” Non-blocking data load with loading indicator
- рҹ“Ң **Status Bar** вҖ” Shows current section, record count and last action
- рҹ”ғ **Column Sorting** вҖ” Click any column header to sort the grid
- вҸі **Loading Indicator** вҖ” UI disabled during data fetch with visual feedback

## вҡҷпёҸ Design Patterns Used

| Pattern | Where Used |
|---|---|
| **2-Project Architecture** | Core (logic) + App (UI) separation |
| **Interface / Contract** | IMenuItemService, ICustomerService, IOrderService |
| **Constructor Injection** | Services injected into Views via MainForm |
| **Shell + Content Pattern** | MainForm as shell, UserControls as pages |
| **View Caching** | Dictionary<Type, UserControl> in MainForm |
| **Repository Pattern** | DB Services implement interface contracts |
| **DRY Principle** | MapRow() helper in every DB service |
| **Mode-Driven Forms** | Add / Edit / View modes in all forms |

## рҹҡҖ Getting Started

### Prerequisites
- Visual Studio 2022 or later
- .NET 10 SDK
- SQL Server 2017 or later (or LocalDB)
- SQL Server Management Studio (optional)

### Installation

**Step 1 вҖ” Clone the repository**
```bash
git clone https://github.com/YourUsername/FastBite-POS.git
cd FastBite-POS
```

**Step 2 вҖ” Set up the database**
- Open SQL Server Management Studio
- Connect to your SQL Server instance
- Open `FastBite_DB_Schema.sql`
- Press F5 to execute
- Database `FastBitePOS` will be created with seed data

**Step 3 вҖ” Update connection string**

Open `FastBite.App/Forms/MainForm.cs` and update:
```csharp
private readonly string _connStr =
    "Server=YOUR_SERVER_NAME; Database=FastBitePOS;" +
    "Integrated Security=True; TrustServerCertificate=True;";
```

Replace `YOUR_SERVER_NAME` with your SQL Server instance name.

**Step 4 вҖ” Restore NuGet packages**
```bash
dotnet restore
```

**Step 5 вҖ” Build and run**
```bash
dotnet build
dotnet run --project FastBite.App
```

Or press **F5** in Visual Studio.
